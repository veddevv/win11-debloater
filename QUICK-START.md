## üîß Troubleshooting

### Cannot run PowerShell scripts

- Right-click ‚Üí "Run with PowerShell" (not double-click)
- Or run PowerShell as admin first

### Not recognized as administrator

- Right-click PowerShell ‚Üí "Run as Administrator"
- Then run: `.\win11-debloater-gaming.ps1`

### Execution policy error

- Run: `Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass -Force`
- Then run the script again

### Want to undo changes?

- Search "Create a restore point"
- Click "System Restore"
- Select the restore point
- Follow wizard
