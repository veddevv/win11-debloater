# win11-deboater

Interactive scripts to remove bloatware from Windows 11. Choose exactly what to remove with simple Y/N prompts.


## Features

- **Interactive** - Choose what to remove with Y/N prompts
- **Safe** - Option to create system restore point
- **Two versions** - PowerShell (.ps1) and Batch (.bat)
- **Color output** - Easy-to-read console (PowerShell)

## What Can Be Removed

**Apps:** Xbox, OneDrive, Teams, Cortana, Mail, Weather, Maps, Photos, Calculator, Music, Movies, Paint 3D, Skype, Sticky Notes, Your Phone, Games (Solitaire, Candy Crush), and more

**System:** Telemetry, Windows Tips, Bing Search, Activity History, Store Auto-Updates

## Usage

### PowerShell (Recommended)

Right-click `windows11-debloat.ps1` → **Run with PowerShell** → Allow admin → Follow prompts

### Batch

Right-click `windows11-debloat.bat` → **Run as administrator** → Follow prompts

## Warning

- **Run as Administrator** required
- **Recommended:** Create restore point when prompted
- **Backup data** especially OneDrive files
- Some apps may be difficult to reinstall

## Undo Changes

Search "Create a restore point" → **System Restore** → Select restore point
